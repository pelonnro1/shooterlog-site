# Data Storage and Backup Terms

Effective date: 2026-03-06  
Version: 2026-03-06-v4

## 1. Scope

These terms describe local storage and backup handling in ShooterLog.

This build does not use mandatory backend account sync.

## 2. Local Data

- Sessions, templates, gun records, and linked images are stored on your device.
- You can export and restore backup files from Settings.
- Restoring a backup replaces the current local data.

## 3. Backup Files

- Backup export creates one file that can include app data and image payloads.
- You choose where to store backups (for example Drive/iCloud/OneDrive).
- Restoring a backup file replaces the current local data.
- Anyone with access to a backup file may be able to access your data.

## 4. User Responsibility

- Protect your device and backup destinations.
- Delete backups you no longer need.
- Verify restore results after importing a backup.

## 5. Third-Party Services

If you store backups in third-party services, their privacy/security terms apply. ShooterLog does not control those policies.

## 6. Deletion and Contact

- You can delete local data in-app.
- You can delete backups from your chosen storage provider(s).
- Contact: `privacy@shooterlog.app`
